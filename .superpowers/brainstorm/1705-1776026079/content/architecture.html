System Architecture

How the pieces fit together end-to-end

Input

Your CV + Campaign Config

  • Upload PDF → AI extracts profile
  • Set location (city / country / remote)
  • Set field (SWE, DS/ML, or both)
  • Optional: paste company list
→

Discovery Engine (Hybrid)

🏢 Active Path

Adzuna / SerpAPI scrape job boards for companies actively hiring interns in your field + location

🔍 Cold Path

AI agent searches web for relevant companies NOT actively posting — startups, labs, scale-ups

Contact Finder

Find the right person

  • Recruiter/HR first (LinkedIn search)
  • Engineer/lead if no recruiter found
  • Email via Hunter.io API
  • LinkedIn profile URL
→

Draft Generator

AI writes per-contact

  • Email: subject + body (formal)
  • LinkedIn DM (short, warm)
  • Tailored to company + role + your CV
  • Streams in real-time as it generates

Review UI

Campaign board — all discovered companies with status (draft ready / sent / skipped)
Per-company panel — contact info, email draft, LinkedIn draft, edit inline or regenerate
Send bar — select all / pick & choose → send via Gmail OAuth or copy LinkedIn DM

Stack

⚡ Next.js 15 App Router
🤖 Vercel AI SDK + Claude claude-sonnet-4-6
🗃️ Supabase (Postgres)
🎨 Tailwind + shadcn/ui
📧 Gmail OAuth for sending
🔍 Hunter.io email finder
🌐 Adzuna API job boards
